# 后台管理系统 - 长期类型项目设计需求文档
**文档版本**：V1.2
**适用模块**：项目管理模块 - 长期项目
**开发说明**：基于现有系统逻辑开发，专属长期项目类型定制，完全兼容系统现有规范，不破坏原有业务功能

---

## 一、需求概述
本次需求针对**长期类型项目**进行全流程专属设计，包含基础字段固定隐藏、自定义项目状态、项目周期自动计算、资金区域文案修改、子项目管理模块开发、成本/财务数据联动计算，所有交互、样式沿用系统现有规范。

---

## 二、核心基础规则
长期项目基础信息中，以下字段**始终隐藏、不占位、不展示**：
施工周期、回款周期、人员数量、订单金额

---

## 三、长期项目 - 新建页面逻辑（项目类型选择「长期」时生效）
### 3.1 基础项目信息
1. **项目状态专属配置**
   - 长期项目仅支持两个状态：`合作中`、`已终止`
   - 两种状态新增至系统配置表，**仅长期项目使用**，不影响其他项目类型
   - 新建限制：**仅展示「合作中」状态，默认选中且不可修改**，无「已终止」选项
   - 项目状态为“已终止”, 那么不允许添加新的子项目，也不允许编辑已有的子项目信息
2. **状态与日期/周期联动规则**
   - 新建默认状态**合作中**：显示「开始日期」，隐藏项目周期；开始日期可选择≤当前系统日期
   - 创建成功后：隐藏「开始日期」，显示**项目周期**（禁止手动编辑）
     - 开始日期 = 创建时选择的日期（固定不变）
     - 状态`合作中`：结束日期**每日自动更新为当前日期**
     - 状态`已终止`：结束日期**停止更新、固定锁定**
     - 状态切回`合作中`：结束日期**恢复自动更新**
   - 状态规则：长期项目**支持状态回溯切换**（仅编辑页可用）
3. **通用字段**：客户名称、客户角色、客户来源、是否有合同、是否有预览图、项目描述，逻辑不变
4. **新增展示**：基础信息中**展示「已收总账款」输入区域**，用于录入更新最新收款额

### 3.2 项目资金管理区域
1. **文案修改**：所有金额字段中间增加**总**字
   - 订单金额 → 订单总金额（只读，子项目汇总）
   - 已收账款 → 已收总账款（只读, 同步基础信息中”已收总账款“填写金额）
2. **计算与联动规则**
   - 订单总金额 = 所有子项目订单金额总和（只读展示）
   - 已收总账款：只读, 同步基础信息中”已收总账款“填写金额，修改后**实时同步更新**项目资金管理、财务快报区域数据

### 3.3 成本支出管理区域
1. 隐藏系统原本的**主项目成本支出管理**区域
2. 成本支出功能迁移至**子项目**中，每个子项目绑定独立的成本支出填写区域

### 3.4 项目合同/预览图管理
- 逻辑、样式、交互完全不变
- 文件与总项目ID绑定，沿用原有上传、存储、删除规则

### 3.5 子项目管理区域（新增核心模块）
位于页面最下方，支持多子项目添加，字段配置如下：
1. **项目内容**：单选框，选项固定：植物养护、植物换新
2. **开始日期**：日期选择器，仅可选择≤当前日期
3. **订单金额**：数字输入框
4. **成本支出**：独立区域，样式参考原有成本支出管理，与当前子项目绑定；可不填写，默认成本为0
5. **操作按钮**：点击「添加子项目」按钮，新增一组子项目填写区域

### 3.6 财务快报区域
- 数据来源：项目资金管理区域数据
- 已收总账款修改后，**财务快报实时重新计算更新**
- 样式、逻辑与原有系统一致

### 3.7 方案预览图区域
逻辑、样式、交互完全保持不变

### 3.8 成本构成分析区域
- 数据来源：所有子项目的成本支出数据总和
- 展示样式、图表逻辑与原有系统一致

## 四、长期项目 - 编辑页面逻辑
1. **项目状态**：同时显示「合作中」「已终止」，支持自由回溯切换
2. **子项目权限**：仅编辑以及新增的时候允许新增子项目
3. **已收总账款**：仅在”项目基础信息管理“区域支持编辑修改，修改后全页面数据实时联动更新
4. **基础编辑权限**：其余规则与新建页面完全一致
5. **项目周期**：延续自动更新/锁定规则，禁止手动编辑

---

## 五、交互与样式要求
1. **固定隐藏字段**：施工周期、回款周期、人员数量、订单金额永久隐藏
2. **只读样式**：项目周期、订单总金额为只读样式，无编辑入口
3. **可编辑样式**：已收总账款使用系统标准输入框样式，支持修改
4. **实时更新**：已收总账款修改后，资金区域、财务快报**无延迟刷新计算**
5. **日期校验**：所有开始日期严格限制≤当前日期
6. **状态展示**：新建仅显示合作中，编辑页显示全部两个状态

---

## 六、开发注意事项
1. **状态配置**：新增合作中、已终止状态至配置表，仅长期项目可用
2. **状态权限**：新建页隐藏已终止，编辑页展示全部状态
3. **自动更新**：合作中状态每日自动更新项目周期结束日期
4. **数据联动**：已收总账款修改 → 实时更新资金、财务快报数据
5. **数据绑定**：子项目、总金额、成本与总项目ID强关联
6. **状态回溯**：仅长期项目支持状态自由切换

---

## 七、核心开发要点
1. **固定隐藏**：长期项目隐藏施工/回款周期、人员数量、订单金额
2. **专属状态**：新建默认合作中（不可选），编辑可切换已终止
3. **周期规则**：开始日期固定，合作中自动更新结束日期
4. **资金核心修改**
   - 基础信息展示**已收总账款**（可编辑）
   - 已收总账款修改 → 全页面实时计算更新
5. **金额规则**：订单总金额=子项目汇总（只读），已收总账款统一在基础信息维护
6. **成本迁移**：主项目隐藏成本管理区域，子项目绑定独立成本区域
7. **编辑限制**：仅编辑页可新增子项目、切换状态
